# TimeTracker ⏱️

**TimeTracker** — это консольное приложение для персонального учёта рабочего времени.  
Позволяет вести список задач, запускать и останавливать таймер.


## Функционал

- Добавление новой задачи с произвольным именем;
- Редактирование и удаление задач;
- Просмотр списка всех задач с накопленным временем (часы, минуты, секунды);
- Запуск таймера для выбранной задачи;
- Остановка таймера;
- Автоматическое сохранение состояния в `data.json` после каждого действия;
- Загрузка предыдущего состояния при запуске программы;
- Отчёт за сегодня;
- Отчёт за период.

## Зависимости

- Компилятор с поддержкой **C++17**;
- **CMake** версии 3.10 или выше;
- Библиотека [nlohmann/json](https://github.com/nlohmann/json) (в проекте `json.hpp`).


## Сборка

Из корня проекта выполнить:

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

```bash
.\build\bin\TimeTracker.exe
```
